
This project is a modified version of the Forge mdk for Minecraft 1.20.1.

The following changes were made to the mdk:
1. ExampleMod renamed to TutorialMod
2. Java package name changed from com.example.examplemods to net.javanoodle.TutorialMod
3. Tutorial Mod properties changed in the gradle.properties file
4. Parchment configured for property mapping

Steps for getting this project up and running can be found on the Minecraft Forge site here:
https://docs.minecraftforge.net/en/latest/gettingstarted/

Specifically, steps 3 and 4 can be followed after downloading the code.
